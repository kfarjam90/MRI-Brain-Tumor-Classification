Enhanced MRI Brain Tumor Classification

Upload an MRI brain image to classify it as normal or identify tumor types (glioma, meningioma, pituitary)

About the Project

This web application uses a deep learning model (ResNet50V2) trained on a dataset of MRI brain images to classify them into four categories:

  • Glioma: A type of tumor that occurs in the brain and spinal cord
  • Meningioma: A tumor that arises from the meninges
  • Pituitary: Tumors that develop in the pituitary gland
  • Normal: No tumor detected

The model was trained to ensure high performance even on low-quality images, promoting excellent generalization for diverse healthcare settings.
